>> 트랙제 소개
디지털소프트웨어공학부는 컴퓨터공학전공, 소프트웨어전공, 사이버보안전공이 통합되어 23학번부터 신설된 전공이다. 이에 따라 트랙제가 도입되었는데, 총 5개 트랙, 빅데이터/웹&앱/인공지능/사물인터넷/사이버보안으로 나뉜다. 각 트랙은 주제별 분류일 뿐 반드시 트랙을 정해서 수강할 필요는 없다.

| 웹&앱 트랙

여러 소프트웨어 기술 분야를 실제로 사용할 수 있도록 구현하는 트랙



> HCI(2-2)
human-computer-interaction의 약자로, 사람들이 컴퓨터 시스템과 어떻게 상호작용하는지 연구한다.


> 파이썬 웹프로그래밍(2-2)
파이썬을 이용해서 웹 애플리케이션을 개발하는 방법을 배우는 과정이다. 데이터 구조(리스트, 딕셔너리, 튜플 등)를 기반으로 웹을 설계할 수 있도록 한다.


> 자바 웹프로그래밍(3-1)
객체지향언어로서 자바 언어의 문법 구조 등을 학습하고 자바 프로그램을 작성하는 능력을 함양한다. 자바 언어는 다른 언어와 달리 한 번 작성하면 모든 컴퓨터, 모든 운영체제에 상관없이 실행 가능한 장점을 갖고 있어 데스크탑, 웹, 모바일 등 다양한 환경에서 사용된다. 필수적인 컬렉션을 배우고, Swing을 이용해 GUI 프로그램을 작성하는 방법을 다룬다.


> 안드로이드 프로그래밍(3-1)
자바를 사용하여 안드로이드 프로그래밍을 배운다. “자바객체지향프로그래밍”, “JSP 서버 프로그래밍”의 과목을 수강하였다면 이 강의가 더 수월할 것이다. 웹 서버 연동과 HTML 파싱, 공공 API와 JSON/XML 파싱하는 것을 다룬다.


> 아이폰 프로그래밍(3-2)
스위프트프로그래밍과 iOS프로그래밍, 모바일데이터베이스응용에 대하여 학습한다. 스위프트 언어의 문법과 코딩 방식을 먼저 배우고, 스위프트 언어를 사용한 iOS 프로그래밍 기초기술, 모바일 데이터베이스응용 앱을 구현해서 서버 프로그래밍 과목의 최종 프로젝트와 연동하도록 구현한다.


> 웹프로그래밍 응용
컴퓨터의 동작원리와 응용전반에 대해 학습을 하는 교과목으로, 컴퓨터가 어떻게 개발되었고 활용될 수 있는지에 대한 이론 학습을 통해, 컴퓨터 부팅, 프로그램 실행의 과정, 컴퓨터가 사용하는 언어는 무엇인지, 컴퓨터는 어떻게 방대한 데이터를 처리할 수 있는지, 그리고 인터넷에서 정보는 어떻게 전송되는지 등을 단계별로 학습한다. IT 기술의 비약적인 발전을 통한 IoT, 모바일컴퓨팅, 빅데이터, 인공지능 등 최신 ICT 트랜드에 대해 학습한다.
